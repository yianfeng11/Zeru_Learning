網頁教學

整個網頁執行流程由前端+後端+資料庫所組成，為了讓大家了解這三個部分擔任什麼功能，因此將在後續幾周帶各位寫一點demo，讓大家了解平常在使用的網頁是如何建立的

前端

1. 前端=Html+Css+Javascript

工具準備:Visual studio code

軟體下載
https://code.visualstudio.com/

Html:由<>組成html的tag (人的骨架)

Html可用的tag列表
https://www.w3schools.com/tags/tag_button.asp

CSS :<link>或< style>組成 (衣服)

CSS使用方式有兩種

1.只能對單一標籤進行style設定
< div style="color:#121212"> 

2.寫一個style套用到全部
< style>
    .test1{
        height: 500px;width:500px;background-color: cornsilk;margin-left: 50px
    }
    .test2{
        background-color:#454545
    }
</ style>

< div class="test1">

Javascript:網頁裡面有< script>區域 (動作)

範例檔案NAS上面 /project/內部分享內容/前端教學.html
1.介接停車場資料API 輸入在網頁上
http://moea.datarget.com.tw/hsinchuSmartBusiness/swagger-ui.html#!/停車場/getPlaceListUsingGET_4
2.請增加停車場搜尋範圍以及排序方式 input type
3.更改input type的style (height,background-color…)

介接突發事件 更改input style
http://moea.datarget.com.tw/hsinchuSmartBusiness/events/list?radius=500&lat=25.12263&lng=121.73558

2. 用chrome開啟→右鍵→檢視原始碼

http://163.29.157.129/TwinTaipei2/login
檢視原始碼

後端

  1. python與django環境設定
  2. 寫個api提供給前端介接
